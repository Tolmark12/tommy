Class JavaTraceMethods
 
Packagedelorum.util
Classpublic class JavaTrace

A simple helper class for sending trace messages to javascript when a swf is embedded in an html page.


Example
Sample usage: 
  
 JavaTrace.trace( "My Message!" );
 //Displays: My Message

 JavaTrace.confirm( "Confirm My Message," );
 // Displays: Confirm My Message, and waits for confirmation
